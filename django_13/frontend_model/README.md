# Модель интерфейса интернет-магазина

Эта папка содержит чистую HTML-верстку на Bootstrap 5 для визуального представления конечного интерфейса нашего Django-проекта.

## Назначение
- Показать структуру шаблонов
- Продемонстрировать UI-компоненты
- Определить навигацию между страницами
- Служить референсом при разработке Django-шаблонов

## Особенности
- Используется только Bootstrap 5 (CDN)
- Нет CSS/JS файлов проекта
- Нет Django тегов шаблонизатора
- Статические данные (товары, пользователи)
- Имитация функционала (кнопки, формы)

## Как использовать
Открывайте файлы в браузере как обычные HTML-страницы.
Все страницы связаны между собой через относительные пути.
